## 六、专项优化建议

### 6.1 解决 JS 渲染内容不可见问题（最关键）

**问题**：`detail.html` 和 `article-detail.html` 的核心内容完全由 JS 动态渲染，搜索引擎爬虫（尤其是百度）无法看到。

**方案 A：服务端渲染（SSR）**
- 最佳方案，后端直接输出 HTML
- 每个门店/文章 URL 对应一个完整 HTML 页面
- 适合正式上线阶段

**方案 B：预渲染（Prerender）**
- 使用 prerender.io 或类似服务
- 为爬虫 User-Agent 返回预渲染的 HTML 快照
- 适合快速上线

**方案 C：静态生成（SSG）**
- 构建时生成所有详情页 HTML
- 适合数据不频繁变化的场景
