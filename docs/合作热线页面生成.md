## 五、交互实现

### Tab 切换

```js
// 按钮绑定
tabMerchant → 显示 #merchant-section，隐藏 #customer-section
tabCustomer → 显示 #customer-section，隐藏 #merchant-section

// 选中状态
.active 类添加至当前按钮，aria-pressed 属性同步更新

// 动画
CSS transition: opacity 0.4s, transform 0.4s
显示时：opacity 1, translateY(0)
隐藏时：opacity 0, translateY(12px)
```

### 汉堡菜单

移动端点击汉堡按钮，展开/收起全屏导航覆盖层（`.nav-mobile`）。

### 行动按钮

- 商家合作行动按钮（咨询加盟/了解托管/联营咨询）：`href="#business-inquiry"`，JS 拦截点击弹窗提示拨打热线
- 客户合作申请按钮（申请银卡/金卡/钻石卡）：`href="tel:0771-8888888"`，直接拨打
