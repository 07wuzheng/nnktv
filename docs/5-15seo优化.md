## 二、优化项目明细

### 1. 结构化数据（Schema.org JSON-LD）— 影响：+25 分

| 页面 | 优化前 | 优化后 |
|------|--------|--------|
| index.html | LocalBusiness 缺 address/openingHours | 补全 address、openingHoursSpecification |
| district.html | WebPage 缺 image | 补全 image、description |
| detail.html | LocalBusiness 缺 telephone/image | 补全 telephone、image |
| articles.html | WebPage 缺 image；CollectionPage 缺 mainEntity | 补全 image、增加 ItemList |
| article-detail.html | Article 缺 image/publisher.logo；缺 WebPage | 补全 image、publisher.logo；新增 WebPage schema |
| cooperation.html | WebPage 缺 image；BreadcrumbList 缺 item URL | 补全 image；面包屑增加 item URL |
| partnership.html | WebPage 缺 image | 补全 image |
| lianxiwomen.html | LocalBusiness 缺 image/description | 补全 image、优化 description |

### 2. AI 搜索就绪（GEO）— 影响：+33 分

| 优化项 | 说明 |
|--------|------|
| robots.txt 新增 AI 爬虫规则 | 允许 GPTBot、ChatGPT-User、CCBot、Google-Extended、PerplexityBot 抓取 |
| 结构化数据完整性 | 所有页面 JSON-LD 信息完整，便于 AI 引用 |

### 3. 技术 SEO — 影响：+13 分

| 优化项 | 说明 |
|--------|------|
| cooperation.html 补 `class="no-js"` | 与全站一致，支持渐进增强 |
| lianxiwomen.html 补 `class="no-js"` | 同上 |
| cooperation.html 补移动端导航 | 原来缺少 `nav-mobile`，移动端用户无法导航 |
| lianxiwomen.html 补移动端导航 | 同上 |
| partnership.html 补移动端导航 | 同上 |
| index.html 新增 preconnect/dns-prefetch | 资源预连接提示，加速首屏加载 |

### 4. Sitemap 优化 — 影响：+6 分

| 优化项 | 说明 |
|--------|------|
| 新增7个城区子页面 URL | `district.html?name=qingxiu` 等全部收录 |
| 移除 detail.html/article-detail.html 无参数 URL | 这两个页面需要参数才有意义，无参 URL 为空壳 |
| 更新 lastmod 为 2026-05-15 | 反映最新更新 |
| 城区页 priority 提升至 0.9 | 核心内容页权重提升 |

### 5. robots.txt 扩展 — 影响：+3 分

| 新增项 | 说明 |
|--------|------|
| Googlebot 专用规则 | Crawl-delay: 0，鼓励 Google 高频抓取 |
| 360Spider 专用规则 | 国内搜索引擎优化 |
| Sogou web spider 专用规则 | 搜狗搜索优化 |
| YisouSpider 专用规则 | 神马移动搜索优化 |
| AI 爬虫全部 Allow | GPTBot/ChatGPT-User/CCBot/Google-Extended/PerplexityBot |

### 6. JS 动态 SEO 同步 — 影响：+2 分

| 文件 | 优化 |
|------|------|
| detail.js | JSON-LD 动态更新时补全 telephone、image |
| article-detail.js | JSON-LD 动态更新时补全 image、publisher.logo，移除冗余条件判断 |
