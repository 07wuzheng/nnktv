### 接口：获取热门KTV门店

- **请求方法**：GET
- **端点**：`/api/ktv/featured`

**请求参数**

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| limit | integer | 否 | 返回数量，默认8 |

**响应字段**

| 字段名 | 类型 | 必填 | 说明 | 示例 |
|--------|------|------|------|------|
| code | integer | 是 | 状态码 | `200` |
| data | object | 是 | 数据对象 | — |
| data.list | array | 是 | 门店列表 | — |
| data.list[].id | string | 是 | 门店ID | `"1"` |
| data.list[].name | string | 是 | 门店名称 | `"XX娱乐KTV（青秀店）"` |
| data.list[].district | string | 是 | 城区拼音slug | `"qingxiu"` |
| data.list[].address | string | 是 | 门店地址 | `"南宁市青秀区民族大道XX号"` |
| data.list[].cover | string | 否 | 门店封面图URL（首页卡片、地区列表、详情页共用） | `"https://img.nanningktv.com/store/1.jpg"` |
| data.list[].create_time | string | 是 | 创建时间，用于排序（DESC） | `"2026-05-14 10:00:00"` |

**排序逻辑**：按 `create_time DESC` 排序，每个地区最新上传的门店优先展示。

**响应示例**
```json
{
  "code": 200,
  "data": {
    "list": [
      {
        "id": "1",
        "name": "XX娱乐KTV（青秀店）",
        "district": "qingxiu",
        "address": "南宁市青秀区民族大道XX号",
        "cover": "https://img.nanningktv.com/store/1.jpg",
        "create_time": "2026-05-14 10:00:00"
      }
    ]
  }
}
```

- **请求方法**：GET
- **端点**：`/api/ktv/first-store-covers`

**请求参数**

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| — | — | — | 无参数（一次性返回7个城区） |

**响应字段**

返回一个以城区拼音slug为key的对象：

| 字段名 | 类型 | 必填 | 说明 | 示例 |
|--------|------|------|------|------|
| {district_slug} | object | 是 | 城区对象，key为拼音（如 `qingxiu`） | — |
| {district_slug}.cover | string | 是 | 该区第一家门店的封面图URL | `"https://img.nanningktv.com/store/1.jpg"` |

### 接口：获取门店详情

- **请求方法**：GET
- **端点**：`/api/ktv/detail`

**请求参数**

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| id | string | 是 | 门店ID |
| district | string | 是 | 城区拼音slug |

**响应字段**

| 字段名 | 类型 | 必填 | 说明 | 示例 |
|--------|------|------|------|------|
| id | string | 是 | 门店ID | `"1"` |
| title | string | 是 | 门店名称 | `"青秀区 XX 娱乐场所（1号店）"` |
| time | string | 是 | 发布/更新日期 | `"2026-05-01"` |
| address | string | 是 | 门店地址 | `"南宁市青秀区XX路XX号XX大厦"` |
| package | string | 是 | 套餐信息 | `"XX套餐：XX小时欢唱+XX饮品"` |
| description | string | 是 | 门店描述（支持 `\n` 换行分段渲染） | `"环境优雅\n音响品质XX"` |
| phone | string | 是 | 预订电话 | `"19968122123"` |
| district | string | 是 | 城区拼音slug | `"qingxiu"` |

  "description": "环境优雅\n音响品质XX\n聚会首选场所",
  "phone": "19968122123",
  "district": "qingxiu",
  "images": [
    "https://img.nanningktv.com/store/1-1.jpg",

### 接口：获取文章详情

- **请求方法**：GET
- **端点**：`/api/articles/detail`

**请求参数**

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| id | integer | 是 | 文章ID |

**响应字段**

| 字段名 | 类型 | 必填 | 说明 | 示例 |
|--------|------|------|------|------|
| id | integer | 是 | 文章ID | `1` |
| title | string | 是 | 文章标题 | `"南宁青秀区KTV新开业优惠活动"` |
| seo_keywords | string | 否 | SEO关键词 | `"南宁KTV,青秀区KTV,优惠活动,南宁唱歌"` |
| content | string | 是 | 文章正文（富文本HTML） | `"<p>南宁青秀区近日新开业...</p>"` |
| author | string | 否 | 作者 | `"南宁KTV"` |
| create_time | string | 是 | 创建时间 | `"2026-05-10 14:30:00"` |
| update_time | string | 否 | 更新时间（与 create_time 不同时前端展示） | `"2026-05-11 09:00:00"` |
| site_id | integer | 否 | 站点ID | `1` |
| region_id | integer | 否 | 区域ID | `1` |

| status | integer | 否 | 状态（1=已发布） | `1` |

- `content` 直接以 `innerHTML` 渲染（富文本，后端需做好 XSS 过滤）
- `update_time` 仅在与 `create_time` 不同时展示

## 五、图片字段使用规则

| 字段 | 用途 | 共用场景 | 渲染方式 |
|------|------|----------|----------|
| `cover` | 门店封面图 | 首页热门卡片 → 地区列表卡片 → 详情页主图 | `background-image` + `background-size:cover` |
| `thumbnail` | 文章封面图 | 首页资讯卡片 → 资讯列表卡片 → 详情页文章图 | `background-image` + `background-size:cover` |
| `images[]` | 门店多图 | 仅详情页图片列表 | `<img>` 标签 + `loading="lazy"` |

**占位规则**：当 `cover`/`thumbnail` 为空或不含有效 `http` URL 时，前端显示灰色占位图，不会发起图片请求。

## 六、全局响应格式约定

### 成功响应

门店类接口（热门门店）：
```json
{ "code": 200, "data": { "list": [...] } }
```

列表类接口（城区门店、资讯列表）：
```json
{ "list": [...], "total": 23, "page": 1, "pageSize": 15 }
```

详情类接口（门店详情、文章详情）：
```json
{ "id": 1, "title": "...", ... }
```

### 错误响应

```json
{ "code": 400, "msg": "参数错误" }
```

| code | 说明 |
|------|------|
| 200 | 成功 |
| 400 | 参数错误 |
| 404 | 资源不存在 |
| 500 | 服务器错误 |
