# 南宁KTV网站 — 前次优化操作总结

> 文档生成时间：2026-05-13  
> 涵盖范围：资讯卡片重构 + 手机端优化（第一轮）

---

## 一、资讯卡片重构

### 需求
1. 移除主页"正在加载最新资讯"加载指示器
2. 将4个新闻卡片改为3个资讯卡片 + 1个"查看更多"卡片
3. 确保图片逻辑一致性：主页封面图 = 资讯列表页封面图 = 详情页文章图（同一 `thumbnail` 字段）
4. 遵循 SEO 优化规则
5. 代码做好后端 API 对接准备

### 修改文件

| 文件 | 修改内容 |
|------|----------|
| `index.html` | 移除 `#newsLoading` 元素和底部"查看更多"按钮；改为 3+1 结构（3资讯 + 1"查看更多"卡片）；添加3个 SEO 兜底静态卡片 |
| `js/main.js` | `fetchArticles()` limit 从4改为3；`renderNewsCards()` 渲染3张资讯卡 + 固定"查看更多"卡；`loadArticles()` 移除 newsLoading 引用 |
| `css/style.css` | 替换 `.news-more-wrap`/`.news-more-btn` 样式为 `.news-card-more` 系列（虚线边框、圆形箭头图标） |

### 后端 API 对接说明
- 接口：`GET /api/articles?type=news&limit=3`
- 字段：id / title / thumbnail / seo_keywords / content / author / create_time / update_time / site_id / region_id / status
- 图片逻辑：`thumbnail` 字段三处共用，CSS `background-size: cover` 自动等比缩放

4. 右上角显示电话号码（替代汉堡菜单按钮）

6. URL 不含中文，使用拼音命名（如 '青秀区' → pinyin）

### 关键技术方案

1. **底部导航栏**：5项（首页🏠/KTV🎤/资讯📰/联系📞/合作🤝），桌面端 `display:none`，移动端 `display:block`，使用 `env(safe-area-inset-bottom)` 适配 iPhone 刘海屏
2. **区域卡片**：从 `flex-direction: column-reverse` + `aspect-ratio: 1` 改为 `grid-template-columns: repeat(2, 1fr)` + `aspect-ratio: 16/9`，使用 `display: contents` 扁平化 `.district-row` 和 `.row-2`/`.row-1`
3. **URL 拼音**：区域名映射为拼音 slug（qingxiu/xingning/xixiangtang/liangqing/jiangnan/yongning/wuming）
4. **`data-page` 属性**：底部导航项使用 `data-page` 标识，JS 根据路径匹配高亮

### 遇到的问题及修复

1. **index.html 重复底部导航栏**：批量脚本在已有 bottom-nav 的文件上重复添加，通过搜索计数检测并手动删除
2. **3个文件遗漏底部导航栏**：lianxiwomen.html / partnership.html / cooperation.html 没有匹配 `<script src="js/` 模式，手动添加
3. **CSS 重复 `.nav-mobile-phone` 块**：底部导航样式被追加到重复的块中，通过精确匹配替换修复
