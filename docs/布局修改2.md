## 四、地区背景图上传支持

每张区域卡片的图片区支持通过 **CSS 变量 `--district-bg`** 自定义背景图片。

### 使用方法

在 `.district-home-img` 元素上添加 `style` 属性：

```html
<div class="district-home-img" style="--district-bg: url('images/qingxiu.jpg');">
    <div class="district-home-img-placeholder" aria-hidden="true">🏙️</div>
</div>
```

### 实现原理

```css
.district-home-img {
    background-image: var(--district-bg, none);
    background-size: cover;
    background-position: center;
    background-repeat: no-repeat;
}
```

- 未设置 `--district-bg` 时，显示默认 emoji 占位符
- 设置后，图片区自动切换为覆盖背景图
- 所有 7 区均已添加注释说明，方便后期上传

### 图片尺寸建议

建议上传 **400×300px** 以上的 JPG/WebP 图片，存放在 `images/` 目录。

## 五、电话号码去除交互反馈

根据 SEO 用户体验优化，电话号应**仅展示文本**，不触发任何点击反馈：

### 顶部栏

```css
.top-bar a.phone-number {
    pointer-events: none;   /* 禁用点击 */
    cursor: default;        /* 禁用手指光标 */
    text-decoration: none;  /* 隐藏下划线 */
}
```

### 页脚

```html
<!-- href 替换为 span，指针事件禁用 -->
<span class="phone-number" style="pointer-events:none;">0771-1234567</span>
```

### 影响范围

| 位置 | 修改方式 |
|:----|:---------|
| 顶部栏预订热线 | `a` 添加 `pointer-events: none` |
| 顶部栏合作热线 | `a` 添加 `pointer-events: none` |
| 页脚预订热线 | `a` 改为 `span` + `pointer-events: none` |
