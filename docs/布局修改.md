### 改造前

```
┌────────────────────────────────┐
│       南宁各区 KTV              │
│   ┌──────────────────────┐     │
│   │ 青秀区 KTV  [3家]     │     │
│   │ ┌────┐┌────┐┌────┐ │     │
│   │ │卡1 ││卡2 ││卡3 │ │     │
│   │ └────┘└────┘└────┘ │     │
│   └──────────────────────┘     │
│   ...（重复7个区）              │
└────────────────────────────────┘
```

### 改造后

```
┌────────────────────────────────┐
│   南宁KTV · 选择区域           │
│   ┌─────────────────────────┐ │
│   │ 青秀区           ┌────┐ │ │
│   │ 南宁CBD...       │ 🏙️ │ │ │
│   │ 查看该区KTV →    └────┘ │ │
│   └─────────────────────────┘ │
│   ┌─────────────────────────┐ │
│   │ 兴宁区           ┌────┐ │ │
│   │ 南宁老城...       │ 🏛️ │ │ │
│   │ 查看该区KTV →    └────┘ │ │
│   └─────────────────────────┘ │
│   ...（共7区，整行可点击）      │
└────────────────────────────────┘
```

### 关键变化

1. **排版改为左信息 + 右图片**
   - 左侧：区名（h3）、区域描述、跳转链接
   - 右侧：图片占位区（后期替换为真实图片）
   - 整张卡片是一个 `<a>` 标签，点击跳转到 `district.html?name=区名`

2. **标题层级优化**
   - H2：`南宁KTV · 选择区域`
   - H3：各区名称（如`青秀区`）
   - 点击跳转到各区详情页（`district.html?name=青秀区`）

3. **SEO 提升**
   - 核心内容不再是 JS 动态渲染，而是**静态 HTML**
   - 搜索引擎爬虫可直接抓取所有 7 个区的内容
   - 每张卡片含区名、区域描述、跳转链接

## 五、main.js 精简

由于首页的 KTV 区域已改为**静态 HTML 结构**，移除了以下不再需要的代码：

| 移除内容 | 行数 | 说明 |
|---------|:----:|------|
| `DISTRICTS` 常量 | ~5 | 不再需要在首页定义区数据 |
| `loadKTVList()` | ~15 | 首页不再调用 KTV API |
| `renderDistrictKTV()` | ~30 | 首页不再渲染 KTV 卡片 |
| `loadAllDistricts()` | ~18 | 首页不再加载所有区数据 |

保留的功能：
- ✅ `fetchJSON()` 通用请求函数
- ✅ `loadSiteTitle()` 标题加载
- ✅ `loadArticles()` 文章列表加载
- ✅ `loadContact()` 联系电话加载
- ✅ 汉堡菜单交互
- ✅ `escapeHtml()` 防 XSS 工具函数

## 九、v2 更新（2026-05-11 20:41）

### 9.1 区域置顶

**KTV 区域**从 Banner 下方移至页面**最顶部**，用户一进入页面首先看到南宁 KTV 区域选择：

```
页面顺序（v2）：
┌─ 南宁KTV · 选择区域（7区卡片） ← 置顶
├─ Banner 头图
├─ 最新资讯
└─ 页脚
```

### 9.2 三列网格

区域卡片由原来的**单列**布局改为**3 列网格**，桌面端一行显示 3 个区域：

| 断点 | 列数 | 卡片布局 |
|:----|:----:|:---------|
| ≥1024px | **3 列** | 左信息 + 右图片 |
| 768~1024px | **2 列** | 左信息 + 右图片 |
| <768px | **1 列** | 图片在上、信息在下 |

### 9.3 CSS 调整

| 对象 | 变更 |
|:----|:-----|
| `.district-home-grid` | 新增 `grid-template-columns: repeat(3, 1fr)` |
| 所有卡片内边距 | 从 32px/28px 缩减至 24px/20px |
| 图片区宽度 | 从 200px 缩减至 150px |
| 区名字号 | 从 1.6rem 缩减至 1.3rem |
| 响应式 1024px | 新增 2 列断点 |
| 响应式 768px | 网格改为 1 列 |
