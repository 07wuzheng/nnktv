## 一、优化清单

### 1. H1 标题缩小 0.5 倍

**问题**：首页 H1「南宁KTV — 覆盖青秀区、兴宁区…」在手机端仍为 `2rem`（约 30px），挤压卡片空间。

**根因**：CSS specificity 覆盖失败。基类选择器 `.section-ktv-home .section-title h1`（优先级 0,2,1）高于移动端通用规则 `.section-title h1`（优先级 0,1,1），导致移动端 `font-size: 0.72rem` 从未生效。

**修复**：在 `@media (max-width: 768px)` 中添加同优先级选择器：

```css
.section-ktv-home .section-title h1 {
    font-size: 1rem;      /* 2rem × 0.5 = 1rem */
}
```

480px 断点：

```css
.section-ktv-home .section-title h1 {
    font-size: 0.9rem;
}
```

**文件**：`css/style.css`

---

### 2. 7 个地区卡片调大适配全屏

**问题**：卡片使用 `aspect-ratio: 16/9` 固定比例，无法撑满剩余视口高度，底部留白过多。

**修复**：

```css
/* grid 行均分高度 */
.section-ktv-home .district-home-grid {
    grid-template-rows: repeat(4, 1fr);
    gap: 10px;
}

/* 取消固定宽高比，让 grid 控制高度 */
.district-home-card {
    aspect-ratio: auto;
}
```

**效果**：7 张卡片（4 行 × 2 列 + 第 4 行 1 张居中）均匀填满 header 以下、热门门店区域以上的全部空间。

**文件**：`css/style.css`

### 5. 底部导航栏 KTV → 定位到热门 KTV 门店

**问题**：点击底部导航「KTV」按钮，链接为 `index.html#featuredStores`，但在首页点击时浏览器可能不重新滚动。

**修复**：在 `js/main.js` 中添加点击处理：

```javascript
var ktvNavItem = document.querySelector('.bottom-nav-item[data-page="ktv"]');
if (ktvNavItem) {
    ktvNavItem.addEventListener('click', function(e) {
        var isIndex = path.indexOf('index') !== -1 || path === '/' || path.endsWith('/');
        if (isIndex) {
            e.preventDefault();
            document.getElementById('featuredStores')
                .scrollIntoView({ behavior: 'smooth', block: 'start' });
            history.replaceState(null, '', '#featuredStores');
        }
    });
}
```

同时添加 `scroll-margin-top` 避免 sticky header 遮挡：

```css
#featuredStores {
    scroll-margin-top: 70px;
}
```

**文件**：`js/main.js`、`css/style.css`

## 二、URL 中文转拼音规范

项目已遵循「URL 不含中文」规则，所有区域名称使用拼音 slug：

| 中文名 | URL slug | 示例 |
|--------|----------|------|
| 青秀区 | `qingxiu` | `district.html?name=qingxiu` |
| 兴宁区 | `xingning` | `district.html?name=xingning` |
| 西乡塘区 | `xixiangtang` | `district.html?name=xixiangtang` |
| 良庆区 | `liangqing` | `district.html?name=liangqing` |
| 江南区 | `jiangnan` | `district.html?name=jiangnan` |
| 邕宁区 | `yongning` | `district.html?name=yongning` |
| 武鸣区 | `wuming` | `district.html?name=wuming` |

**JS 映射表**（`main.js` 中的 `DISTRICT_MAP`）：

```javascript
var DISTRICT_MAP = {
    qingxiu:    { name: '青秀区' },
    xingning:   { name: '兴宁区' },
    jiangnan:   { name: '江南区' },
    xixiangtang:{ name: '西乡塘区' },
    liangqing:  { name: '良庆区' },
    yongning:   { name: '邕宁区' },
    wuming:     { name: '武鸣区' }
};
```

所有动态生成的 URL 均通过 `DISTRICT_MAP` 的 key 构造，确保 URL 中不含中文：

- 门店详情：`detail.html?district=qingxiu&id=1`
- 区域列表：`district.html?name=qingxiu`
- 文章详情：`article-detail.html?id=1`
