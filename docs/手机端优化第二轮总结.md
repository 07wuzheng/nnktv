# 南宁KTV网站 — 手机端优化（第二轮）操作总结

> 文档生成时间：2026-05-13  
> 涵盖范围：H1字体适配 / 电话号码缩小 / 底部导航KTV定位 / 资讯卡片2列布局 / 联系我们页面精简

---

## 一、H1 标题进一步缩小

### 需求
iPhone 14 Pro Max 仿真设备下 H1 标题仍然偏大，需要进一步缩小。

### 修改
**文件**：`css/style.css`

| 断点 | 修改前 | 修改后 |
|------|--------|--------|
| ≤768px | `0.85rem` | `0.72rem` |
| ≤480px | `0.8rem` | `0.65rem` |

### CSS 代码位置
```css
/* 768px 断点 */
.section-title h1, .section-title h2 { font-size: 0.72rem; }
/* 480px 断点 */
.section-title h1, .section-title h2 { font-size: 0.65rem; }
```

---

## 二、顶部右上角电话号码缩小

### 需求
移动端右上角电话号码字体过大，需要缩小适配。

### 修改
**文件**：`css/style.css`

| 元素 | 修改前 | 修改后 |
|------|--------|--------|
| `.header-phone`（768px） | `font-size: 0.82rem; padding: 4px 10px` | `font-size: 0.7rem; padding: 4px 8px` |
| `.header-phone .phone-number-lg`（768px） | 仅 `font-weight: 700`（继承2.2rem） | `font-size: 0.75rem; font-weight: 700` |

### 说明
原来 `.phone-number-lg` 在桌面端为 `2.2rem`，移动端未覆盖导致显示过大。现移动端单独覆盖为 `0.75rem`。

---

## 三、底部导航 KTV 定位到热门KTV门店

### 需求
底部导航栏点击"KTV"应定位到热门KTV门店页面，而非区域列表。

### 修改
所有8个 HTML 文件中底部导航栏的 KTV 链接：

| 修改前 | 修改后 |
|--------|--------|
| `href="index.html#ktv-list"` | `href="index.html#featuredStores"` |

### 涉及文件
`index.html` / `articles.html` / `article-detail.html` / `district.html` / `detail.html` / `lianxiwomen.html` / `partnership.html` / `cooperation.html`

### JS 高亮逻辑优化
**文件**：`js/main.js`

底部导航栏高亮逻辑新增 hash 检测：

```javascript
// 首页但 hash 指向热门KTV门店区域，高亮 KTV
if (hash === '#featuredStores' || hash === '#ktv-list') {
    page = 'ktv';
}
```

当用户从其他页面点击底部 KTV 导航进入首页 `#featuredStores` 锚点时，底部导航栏自动高亮 KTV 项。

### CSS 补充
**文件**：`css/style.css`

移动端（≤768px）新增热门门店区域 padding 优化：

```css
.section-featured {
    padding: 12px 0 16px;
}
```

## 四、手机端资讯页面一行两个卡片

### 需求
手机端资讯列表页卡片太宽，改为一行两个卡片显示。

### 修改
**文件**：`css/style.css`

480px 断点下 `.articles-page-grid` 布局：

| 修改前 | 修改后 |
|--------|--------|
| `grid-template-columns: 1fr` | `grid-template-columns: repeat(2, 1fr)` |
| `gap: 12px` | `gap: 10px` |
| `.articles-page-card { height: 220px }` | `.articles-page-card { height: 180px }` |

### 说明
768px 断点已经是 `repeat(2, 1fr)` 两列布局，480px 断点原来是单列，现统一改为两列。卡片高度从 220px 缩减至 180px，确保小屏幕下两列卡片不会过于拥挤。

## 五、联系我们页面删除"到店指引"至"品牌宣传语"区域

### 需求
联系我们页面从 `class="section-header"`（到店指引标题）到 `class="brand-slogan"`（品牌宣传语）整个区域全部删除，PC端和移动端均不需要。

### 修改
**文件**：`lianxiwomen.html`

#### HTML 删除
完整删除以下区域：
1. `<section class="direction-section" id="store-direction">` — 到店指引整个区块（地图占位 + 4种交通方式）
2. `<div class="brand-slogan">` — 品牌宣传语

#### CSS 删除
完整删除以下样式及其响应式版本：
- `.direction-section` 及其子元素样式
- `.map-placeholder` 地图占位样式
- `.direction-list` / `.direction-item` 交通指引列表样式
- `.brand-slogan` 品牌宣传语样式
- 768px / 480px 断点下上述元素的响应式样式
