# 5/16 分区整改记录

## 一、整改背景

原网站按南宁市7个行政区域（青秀区、兴宁区、西乡塘区、良庆区、江南区、邕宁区、武鸣区）组织KTV内容，现统一改为按KTV类型分为4大分区：**游戏、商务、高端量贩、花场**。

## 三、布局调整

- **首页分区卡片**：从3列7卡 → 2列4卡（2×2网格）
- **CSS Grid**：`grid-template-columns: repeat(2, 1fr)`
- **移动端全屏分区**：`grid-template-rows: repeat(2, 1fr)`（原4行）
- **多余3个分区直接删除**，未做301跳转（静态站无服务端重写能力）

## 四、SEO 相关

- 所有页面 `<title>`、`<meta description>`、`<meta keywords>`、OG/Twitter 标签已同步更新
- JSON-LD 结构化数据中 `areaServed` 字段更新为 `["游戏", "商务", "高端量贩", "花场"]`
- Sitemap.xml 重写为4个分区URL
- URL 中不含中文，全部使用拼音 slug

## 五、遗留问题

1. **首页导航栏与其他页面不一致**：首页搜索栏位于轮播图下方独立区域，其他页面搜索栏嵌入导航栏内，导致首页导航栏结构与其他页面不同
2. **部分页面 description 仍残留旧文案**：如 `articles.html` 的 description 中仍提到"覆盖七城区"
3. **首页注释仍残留旧文案**：如行185仍写着"南宁 7 区 KTV 首页"
