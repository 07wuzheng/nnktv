# 南宁KTV — SEO优化记录（5/14）

> 优化原则：**不改变任何网站功能交互、页面显示和布局，网站样子保持不变**
> 仅对 SEO 相关的不可见元素进行优化

---

## 一、全站导航链接 `title` 属性补全

**问题**：全站8个页面的桌面导航、移动端导航、页脚导航、底部导航栏的所有链接均缺少 `title` 属性，不利于搜索引擎理解链接语义。

**优化**：为全站所有导航链接添加语义化 `title` 属性：

| 链接文字 | title 属性 |
|----------|-----------|
| 首页 | 南宁KTV首页 |
| 南宁KTV | 南宁KTV门店推荐 |
| 最新资讯 | 南宁KTV行业资讯动态 |
| 联系我们 | 联系南宁KTV |
| 合作热线 | 南宁KTV商家合作热线 |

**涉及页面**：index.html、detail.html、district.html、articles.html、article-detail.html、cooperation.html、partnership.html、lianxiwomen.html

## 六、补全 BreadcrumbList 结构化数据

**问题**：partnership.html 和 lianxiwomen.html 缺少面包屑 JSON-LD 结构化数据。

**新增**：

```json
// partnership.html
{
    "@type": "BreadcrumbList",
    "itemListElement": [
        { "position": 1, "name": "首页", "item": "https://www.nanningktv.com/" },
        { "position": 2, "name": "合作热线", "item": "https://www.nanningktv.com/partnership.html" }
    ]
}

// lianxiwomen.html
{
    "@type": "BreadcrumbList",
    "itemListElement": [
        { "position": 1, "name": "首页", "item": "https://www.nanningktv.com/" },
        { "position": 2, "name": "联系我们", "item": "https://www.nanningktv.com/lianxiwomen.html" }
    ]
}
```

## 七、全站添加 `<meta name="format-detection">`

**问题**：移动端浏览器可能无法自动识别页面中的电话号码。

**优化**：为全站8个页面添加 `<meta name="format-detection" content="telephone=yes">`，确保电话号码可被自动识别为可点击链接。
