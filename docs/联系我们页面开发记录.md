# 联系我们页面开发记录

> 日期：2026-05-13

---

## 一、需求概述

为南宁商务KTV官网创建"联系我们"独立页面，提供清晰的联系方式和地址信息，促进咨询与预订，并为 SEO 提供地理位置和联系方式结构化数据。

### 核心要求

1. 文件名使用拼音 `lianxiwomen.html`，URL 不含中文
2. 复用主站全局导航栏和页脚，配色风格与主页一致
3. 电话号码/按钮点击弹出电话弹窗（与 district/detail 页面效果一致）
4. 三列卡片布局展示：客服热线、总部地址、营业时间
5. 包含到店指引和地图占位区域
6. 添加本地商家 JSON-LD 结构化数据
7. 响应式适配：桌面三列、平板两列、手机单列

## 五、电话弹窗交互

### 按钮列表

| 按钮 | 位置 | 触发行为 |
|------|------|----------|
| 📞 立即拨打 | 客服热线卡片 | 弹出电话弹窗 19968122123 |
| 电话号码链接 | 客服热线卡片 | `<a href="tel:19968122123">` 直接拨打 |

### 弹窗逻辑

- 使用 `data-phone` 属性绑定电话号码
- 点击按钮 → 弹窗显示电话号码 + 可点击拨打链接
- 关闭方式：点击"关闭"按钮 / 点击遮罩层
- 与 district / detail / partnership 页面弹窗完全一致

| `<header>` | 页面头部 |
