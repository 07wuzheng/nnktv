# 5/14 手机端页面第三次优化

## 一、Bug修复

### 1. PC首页切回时90%缩小bug
- **问题**：从其他页面导航回首页时，页面缩小至约90%
- **原因**：浏览器 `bfcache` 恢复页面时，`scrollRestoration` 和 `scrollTo` 未正确重置
- **修复**：所有页面添加 `pageshow` 事件监听，`event.persisted` 时强制重置 `transform`、`zoom` 并回到顶部
- **涉及文件**：
  - `js/main.js`
  - `js/detail.js`
  - `district.html`（内联脚本）
  - `partnership.html`（内联脚本）
  - `lianxiwomen.html`（内联脚本）
  - `cooperation.html`（内联脚本）

### 2. float-wechat-icon 点击无反应（全站）
- **问题**：点击微信悬浮图标无法弹出二维码
- **原因**：`.float-wechat-qr` 定位在图标附近，即使 `scale(0)` + `opacity: 0` 仍会拦截点击事件
- **修复**：`.float-wechat-qr` 默认添加 `pointer-events: none`，hover 和 `.wechat-active` 状态恢复 `pointer-events: auto`
- **涉及文件**：`css/style.css`

## 二、新增功能

### 3. header导航栏搜索栏（PC + 手机端）
- **PC端位置**：`nav-desktop` 和 `header-phone` 中间
- **手机端位置**：`logo` 和 `header-phone` 中间，flex自适应撑满
- **功能**：
  - 模糊搜索全站信息（KTV门店地址 + 资讯信息）
  - 搜索结果按重要程度排序：KTV地址 > 资讯信息
  - 支持拼音首字母、部分匹配、关键词匹配
  - 300ms 防抖输入、ESC 关闭、点击外部关闭
  - 搜索结果链接添加搜索栏HTML和`js/search.js`
  - 搜索栏CSS（桌面端+手机端响应式）
  - 模糊搜索JS：支持中文模糊匹配，按权重排序
  - 搜索结果下拉面板
