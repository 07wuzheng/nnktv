## 一、手机端优化

### 1. 首页H1标题大小与热门KTV门店一致

- **问题**：首页 `class="section-title"` 内的 H1（"南宁KTV — 覆盖青秀区…"）字体偏小
- **修复**：768px断点 `.section-ktv-home .section-title h1` 从 `1.3rem` → `1.44rem`（与热门KTV门店 `1.44rem` 一致）；480px断点从 `1.17rem` → `1.3rem`

### 2. 全站联系二维码图片使用 img/wx.png

- **问题**：所有二维码占位均为空 `src=""` 或文字"二维码待提供"
- **修复**：
  - 所有 `float-wechat-qr img` 的 `src` 从 `""` → `"img/wx.png"`
  - 所有页脚 `.qr-img` 占位文字替换为 `<img src="img/wx.png" alt="南宁KTV微信公众号" width="140" height="140">`
- **涉及页面**：index / cooperation / articles / lianxiwomen / partnership / detail / district / article-detail

### 3. 删除合作页面 platform-stats 区域

- **问题**：partnership.html 底部"平台经营数据"区域（class="platform-stats"）显示全0数据，无实际意义
- **修复**：
  - 删除 HTML 中 `<!-- 平台经营数据 -->` 整个 `<div class="platform-stats">` 区块
  - 删除内联CSS中 `.platform-stats` / `.platform-stat-item` 全部样式
  - 删除响应式中 `.platform-stats` 的 grid-columns 规则（768px / 480px）

### 4. 最新资讯页面 articles-page-header 居中

- **问题**：`class="articles-page-header"` 标题左对齐，视觉不平衡
- **修复**：
  - `.articles-page-header` 添加 `text-align: center;`
  - `.articles-page-header h1::after` 下划线居中：`margin: 8px auto 0;`

### 5. float-wechat-qr 改为左侧显示

- **问题**：移动端QR弹窗向上弹出，遮挡图标
- **修复**：768px断点 `.float-wechat-qr` 改为左侧弹出：
  - `right: 100%; top: 50%; bottom: auto; left: auto;`
  - `transform: translateY(-50%) scale(0); transform-origin: right center;`
  - `margin-right: 12px;`
  - hover和wechat-active状态同步更新

## 二、PC端优化

### 1. 删除导航栏和页脚的"招聘"链接

- **问题**：招聘页面不存在，导航和页脚中的"招聘"链接指向 `#recruitment` 无实际页面
- **修复**：删除所有页面中的招聘链接：
  - 桌面端导航 `nav-desktop` 中的 `<li><a href="#recruitment">招聘</a></li>`
  - 移动端导航 `nav-mobile` 中的招聘链接
  - 页脚 `footer-links` 中的招聘链接
  - 首页页脚的招聘注释
- **涉及页面**：index / articles / cooperation / lianxiwomen / partnership / detail / district / article-detail

### 2. 修复页面切换90%显示bug

- **问题**：点击导航栏切换页面时，部分页面显示在90%位置（未滚动到顶部）
- **原因**：浏览器默认保留上一页的滚动位置（scroll restoration）
- **修复**：
  - 所有JS文件（main.js / detail.js / articles.js / article-detail.js）添加：
    ```javascript
    if ('scrollRestoration' in history) {
        history.scrollRestoration = 'manual';
    }
    window.scrollTo(0, 0);
    ```
  - 内联脚本的页面（cooperation / lianxiwomen / partnership / district）同样添加
  - CSS添加 `html { overflow-y: scroll; }` 确保滚动条始终显示

### 3. SEO优化

- **sitemap.xml**：修复意外截断问题，恢复完整8个URL条目
- **articles.html**：补充 `og:image:width/height`、WebPage JSON-LD 结构化数据
- **article-detail.html**：补充 favicon link
- **district.html**：补充 `og:image:width/height`、Twitter Card meta
- **partnership.html**：修复遗漏的电话号码（钻石卡 `0771-8888888` → `19968122123`，页脚电话同步修复）
