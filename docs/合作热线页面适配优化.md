> 日期：2026-05-13

---

## 一、需求概述

对 `partnership.html`（合作热线页面）进行全面优化，主要包括：

1. 适配主页导航栏与页脚，保持全站结构统一
2. 页面配色风格与主页保持一致（白底蓝字）
3. 全站右上角电话号码加大一倍
4. 客户合作页面会员体系三个卡片按钮对齐至同一水平线
5. 所有操作按钮（咨询加盟、了解托管、联营咨询、申请银卡、申请金卡、申请钻石卡）点击后弹出电话弹窗（与 district/detail 页面效果一致）
6. SEO 规范：URL 无中文、id/class 全英文、中文地名转拼音

## 三、partnership.html 变更详情

### 3.5 无障碍支持

- 添加 `<noscript>` 提示
- 添加 `.skip-link` 跳转链接

### 3.6 会员卡片按钮对齐

**问题**：三张会员卡片（银卡/金卡/钻石卡）按钮不在同一水平线，前两个按钮位置偏高。

**解决方案**：

```css
.member-card {
    display: flex;
    flex-direction: column;
}
.member-benefits {
    flex: 1;
}
.member-card .btn-apply {
    margin-top: 24px;  /* 统一底部间距 */
}
```

- `.member-benefits` 设为 `flex:1`，权益列表自动撑满剩余空间
- 按钮统一 `margin-top: 24px`，保证三张卡片按钮始终在同一水平线

### 3.7 合作模式卡片按钮对齐

同样使用 `display: flex; flex-direction: column;`，按钮加 `margin-top: auto` 实现底部对齐。

### 3.8 按钮点击效果 → 电话弹窗

**修改前**：按钮使用 `<a>` 标签，点击后直接 `alert()` 或 `tel:` 直拨

**修改后**：按钮改为 `<button data-phone="0771-8888888">`，点击后弹出电话弹窗

#### 涉及的 6 个按钮

| 按钮文案 | 所属板块 | 原行为 | 新行为 |
|----------|----------|--------|--------|
| 咨询加盟 | 商家合作 - 品牌加盟 | `#business-inquiry` | 电话弹窗 |
| 了解托管 | 商家合作 - 托管经营 | `#business-inquiry` | 电话弹窗 |
| 联营咨询 | 商家合作 - 联营合作 | `#business-inquiry` | 电话弹窗 |
| 申请银卡 | 客户合作 - 银卡 | `tel:0771-8888888` | 电话弹窗 |
| 申请金卡 | 客户合作 - 金卡 | `tel:0771-8888888` | 电话弹窗 |
| 申请钻石卡 | 客户合作 - 钻石卡 | `tel:0771-8888888` | 电话弹窗 |

#### 弹窗结构

```html
<div id="phoneModal" class="modal-overlay">
    <div class="modal-box">
        <div class="modal-close" id="modalClose">&times;</div>
        <div class="modal-icon">📞</div>
        <h3>咨询热线</h3>
        <a href="tel:0771-8888888" class="modal-phone-link">0771-8888888</a>
        <p class="modal-tip">点击号码即可拨打</p>
    </div>
</div>
```

#### JS 逻辑

```javascript
// 所有带 data-phone 属性的按钮，点击打开弹窗
document.querySelectorAll('[data-phone]').forEach(function(btn) {
    btn.addEventListener('click', function() {
        document.getElementById('phoneModal').style.display = 'flex';
    });
});
// 关闭弹窗
document.getElementById('modalClose').addEventListener('click', function() {
    document.getElementById('phoneModal').style.display = 'none';
});
// 点击遮罩关闭
document.getElementById('phoneModal').addEventListener('click', function(e) {
    if (e.target === this) this.style.display = 'none';
});
```

弹窗效果与 `district.html` / `detail.html` 中的"查看电话"弹窗完全一致。
